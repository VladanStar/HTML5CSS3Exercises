Include Bootstrap in “Most Wanted Criminals” (HTML project) on which you 
worked on at the beginning of the course. 
 
Using downloaded files

poster” page. 
 
CDN 
Use CDN links provided by Bootstrap and include Bootstrap on the “Poster” 
page. 
 
 
 
Basic Grid 
Create a grid with three rows. 
 
The first row should have two columns: 
○width of the first column is 2/3 of the screen width 
○width of the second column is 1/3 of the screen width 
 
The second row should contain three columns: 
○columns should be the same width 
 
The third row should have two columns: 
○both columns should be the same width 
○each row should have a different bg color 
 
All cells in a row should have same bg color, that is different from the row’s 
bg color. 
 
 
 
 
 
Advanced Grid (Responsive) 
Use the grid from the previous example: 
●adapt it to be suitable for a mobile layout 
●extend mobile layout with tablet layout 
●extend layout by adding large display layout
